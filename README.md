## Яндекс Браузер

### Установка из файла (режим разработчика)

1. Скачайте архив с расширением и распакуйте его в отдельную папку, например:  
   `C:\Extensions\corporate-dictionary`

2. Откройте Яндекс Браузер и перейдите на страницу управления расширениями:  
   `browser://extensions/`  
   (или через меню *Настройки → Дополнительно → Расширения*)

3. В правом верхнем углу включите **Режим разработчика**.

4. Нажмите кнопку **Загрузить распакованное расширение**.

5. Выберите папку с распакованным расширением и нажмите **Выбрать папку**.

6. Готово! Иконка расширения появится на панели инструментов.

## Google Chrome

### Установка из файла (режим разработчика)

1. Скачайте архив и распакуйте его, например, в `C:\Extensions\corporate-dictionary`.

2. В адресной строке Chrome введите:  
   `chrome://extensions/`  
   (или меню *Настройки → Расширения*).

3. Включите **Режим разработчика** (переключатель в правом верхнем углу).

4. Нажмите **Загрузить распакованное расширение** и укажите путь к распакованной папке.

5. Подтвердите установку, если потребуется.

## Проверка установки

- ✅ Иконка расширения отображается на панели инструментов браузера.
- ✅ При клике на иконку открывается окно расширения.
- ✅ Добавьте тестовый термин (например, слово `test` с определением `тест`) и откройте страницу, содержащую это слово. Оно должно подсветиться синим, а при наведении — показать определение.

## Устранение проблем

### ❌ Ошибка «Не удалось загрузить распакованное расширение»
**Причина:** Неправильная структура папок или отсутствует файл `manifest.json`.  
**Решение:** Проверьте, что папка содержит все необходимые файлы. Структура должна быть такой:

    corporate-dictionary/
    ├── manifest.json
    ├── background.js
    ├── content.js
    ├── popup.html
    ├── popup.js
    └── icons/
        ├── icon16.png
        ├── icon48.png
        └── icon128.png

## Удаление расширения

### Яндекс Браузер
1. Откройте `browser://extensions/`.
2. Найдите **Corporate Dictionary** и нажмите **Удалить** (или значок корзины).

### Google Chrome
1. Откройте `chrome://extensions/`.
2. Найдите расширение и нажмите **Удалить**.

## Для разработчиков

### Локальная разработка

Клонируйте репозиторий и загрузите расширение в режиме разработчика:

    git clone https://github.com/IvanPro91/corporate_dictionary
    cd corporate-dictionary
    
После внесения изменений не забудьте обновить расширение на странице `chrome://extensions/`.
